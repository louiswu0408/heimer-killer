## Overview

**Heimer Killer** is a small Python/Pygame project where you control a character using the keyboard. Move around, flash, dodge, adjust difficulty, and enjoy the chaos!

---

## How to Run

### 1. Check that Python is installed

Open **PowerShell** and run:

```pwsh
python --version
```

You should see something like:

```
Python 3.x.x
```

If not, install Python first.

---

### 2. Install Pygame

```pwsh
pip install pygame
```

---

### 3. Clone this project

```pwsh
git clone https://github.com/louiswu0408/heimer-killer.git
```

Then go into the folder:

```pwsh
cd heimer-killer
```

---

### 4. Run the game

```pwsh
python "main (3).py"
```

(You can also copy/paste the script provided above into this file.)
